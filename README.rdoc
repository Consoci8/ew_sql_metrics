This rails engine uses Rails 3 Rails Engine & Mongoid. 
It stores all sql operations made by the Rails application into mongodb.
It assumes that the main rails application it's mounted to uses devise or warden for authentication

Check Gemfile for other dependencies.

People can view all sql notifications at /sql_metrics
